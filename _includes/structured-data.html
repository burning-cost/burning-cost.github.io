{% comment %} structured-data.html -------------------- Single source of truth for all schema.org JSON-LD on the site. Site-level entities (every page): Organization — tells Google who runs this site. Conditional entities by layout / front-matter: WebSite — landing layout only (enables SearchAction sitelinks). SoftwareApplication — pages with `schema: SoftwareApplication` in front matter. BlogPosting — post layout. WebPage — page layout. CollectionPage — tags and blog-archive layouts. All page-specific JSON-LD was previously inlined directly in each layout file, which caused duplication when those files also called this include. This include is now the only place structured data is emitted.
